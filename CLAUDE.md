# Davies + Allen — Website Project

You are helping Kimball Mueller (P&L Productions) build a new website for Davies + Allen, an accounting firm with 5 locations in Utah. The site is being built in Go High Level (GHL).

## Client
**Company:** Davies + Allen
**Industry:** Accounting / professional services
**Website (current):** daviesallen.com
**Locations:** 5 offices in Utah

## Project Goal
Rebuild their website in GHL to fix SEO gaps, add location and service pages, and support lead generation. The current site has no keyword-targeted pages, no blog, and no local SEO structure.

## Site Structure
Pages live in `pages/`. Write all copy here first, then build in GHL.

- `pages/home.md` — homepage
- `pages/about.md` — about the firm
- `pages/services/` — one file per service
- `pages/locations/` — one file per office location
- `pages/faq.md` — keyword-targeted FAQ page
- `pages/contact.md` — contact page

## SEO
All page titles, meta descriptions, and keyword targets live in `seo/`.

## Brand
Brand guidelines and asset references in `brand/`.

## Assets
Images, logos, and media references in `assets/`.

## Communication Style
Direct, no fluff. Bullet points and short paragraphs. No em dashes.
Content tone for Davies + Allen: professional, trustworthy, local. Not corporate. Not stiff.
